# 接口动态代理和管控

## 技术范围

* typescript
* koa2

## 解决问题

> 给现有的服务添加可控的动态访问路径
> 如下述场景：
> 客户a想要访问一个现有的服务的接口 /user/nums
> 和客户沟通后，达成条件，开放一年的接口访问权限，一年后如果没有续费，则无法继续访问
> 于是面临的有两套方案
> 方案A，对现有服务改造
> 方案B，提供中间动态接口代理层，在一年内，开放代理层接口 /random/api 代替 /user/nums，一年后自动过期
> 此项目是针对方案B做出的实现
